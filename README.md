# Law Office of Sample & Associates

A professional services website for a fictional law firm in Vacaville, CA. Built as a portfolio demo for the Business Site tier ($1,200) at SS Web Design.

## Live Site

https://law-office-sample.vercel.app/

## About the Project

This is a fictional demo site showcasing a $1,200 Business Site package. It demonstrates professional web design for service-based businesses.

**Fictional Business Details:**
- Name: Law Office of Sample & Associates
- Location: 123 Main Street, Suite 300, Vacaville, CA 95688
- Practice Areas: Family Law, Criminal Defense, Personal Injury, Estate Planning, Business Law
- Founded: 2015

## Tech Stack

- HTML5 (Semantic markup, ARIA labels)
- CSS3 (Custom Properties, Flexbox, CSS Grid)
- JavaScript (Vanilla)
- Mobile-first Responsive Design
- Accessibility-focused (WCAG compliant)

## Features

- 5-page structure: Home, About, Services, Testimonials, Contact
- Responsive navigation with mobile hamburger menu
- Contact form with appointment booking
- Client intake form with file upload
- Google Maps embed
- Social media links
- Google Analytics 4 placeholder
- Legal disclaimers
- Demo mode: Forms display alert instead of submitting

## Pages

| Page | Description |
|------|-------------|
| Home | Hero, practice areas, attorney spotlight, testimonials, CTA |
| About | Firm history, values, 3 attorney bios, awards |
| Services | 5 practice areas with detailed descriptions |
| Testimonials | 6 client reviews, stats banner, legal blog |
| Contact | Office info, map, booking form, intake form |
